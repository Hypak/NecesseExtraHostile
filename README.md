# NecesseExtraHostile
WIP Mod for Necesse.
Makes mobs evil (and some balancing).
Adds Laser Ducks,
Shooting Sheep,
Charging Cows,
Rabid Rabbits,
Cocky Cardinals.
Also scales up the difficulty of hostile mobs (before deep caves).
Adds crafting recipes for enchantment scrolls.
Also makes Evil's Protector Fight faster but harder.
